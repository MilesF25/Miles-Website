Pattern Finder

A very simple file scanner

I wanted to try making a tool that can discover red flags in code like hard coded secrets or red flags in coding. Theres not much backstory besides the fact im looking into securtiy and safe coding.

Flow of the program

my program flows like this. You enter a github repo url, the program goes and gets the contents of the selected files, then it passes it on the the pattern finder and it runs through the code looking for any red flags
